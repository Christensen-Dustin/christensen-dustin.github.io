JavaScript Objects -October 5, 2018--This is a great example of JS but the JS Object is very minimal. A JSO should be a object you can create multiple instances. Your sentencesInput object. Is technically an JSO but is a stand alone JSO. What you should be looking for are ways to create an JSO based on a function. i.e. function Person(first, last, age, eye) { this.firstName = first; this.lastName = last; this.age = age; this.eyeColor = eye; } This takes a simple JSO object (which is often little more than a JSON variable) and makes it a true Object. You can create multiple instances each with their own unique values in the properties..-
